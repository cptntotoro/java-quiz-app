# java-quiz-app

## Основные функции:
- Загрузка Excel-файла с вопросами-ответами 
- Выбор темы из выпадающего списка 
- Листание вопросов по теме 
- Просмотр ответа по клику 
- Отметка "Знаю" для изученных вопросов 
- Статистика по прогрессу изучения 
- Очистка всех данных

## Структура Excel-файла
   
Создайте файл questions.xlsx с тремя колонками:

- Тема (например, "Java Basics")
- Вопрос (например, "Что такое полиморфизм?")
- Ответ (например, "Возможность объектов с одинаковой спецификацией иметь различную реализацию")

Пример:

Тема	Вопрос	Ответ
Java	Что такое инкапсуляция?	Сокрытие внутренней реализации от внешнего доступа
SQL	Что такое PRIMARY KEY?	Уникальный идентификатор записи в таблице

## Инструкция по запуску

### Запуск backend:

```
   cd backend
   mvn spring-boot:run
```


### Запуск frontend:

```cd frontend
   npm install
   npm start
```

## Доступ к приложению:

### Frontend: http://localhost:3000

### Backend API: http://localhost:8080

### H2 Database Console: http://localhost:8080/h2-console
